Just Text

- A light weight note pad application
- Support
    - Cut copy paste
    - Find and Replace
    - Line number
    - Multi tab
    - Internal DB to store notes so explorer to view the notes
    - can save in local file
    - Line function
        - sort lines
        - Remove duplicates
        - Remove empty ,starts with,ends with
    - Simple dark and light theme 
    - Simple executable ,cross platform
    - clipboard copy support
